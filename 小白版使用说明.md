# 小白版使用说明

## 你现在有两个用法

### 用法 A：有 DeepSeek API Key

这个用法最方便，插件会直接在当前网页右下角的小悬浮窗回答。

1. 打开 Chrome。
2. 地址栏输入：`chrome://extensions/`
3. 打开右上角“开发者模式”。
4. 点“加载已解压的扩展程序”。
5. 选择文件夹：`google-question-helper`
6. 点 Chrome 右上角的插件图标，在当前网页右下角打开“搜题学习助手”小悬浮窗。
7. 点插件右上角的设置按钮。
8. 粘贴你的 DeepSeek API Key。
9. 模型先选 `deepseek-v4-flash`。
10. 点“保存设置”。

以后用的时候：

1. 打开有题目的网页。
2. 点 Chrome 右上角的插件图标。
3. 如果挡住题目，可以拖动悬浮窗，或者点“－”最小化。
4. 在浮窗里点“搜题：框选并讲解”。
5. 页面会进入框选模式，拖动鼠标把题目圈起来。
6. 松开鼠标后，插件只识别你圈出来的区域，并在浮窗里给出讲解。

### 备用用法：选中文字

如果当前屏幕识别不准：

1. 在网页上用鼠标选中题目。
2. 右键。
3. 点“把选中题目发到搜题学习助手”。
4. 在浮窗里点“开始讲解”。

## 关于“能不能直接接到 Codex/ChatGPT 这里”

不能稳定地直接接到这个聊天窗口。这个聊天窗口是当前会话，不是给浏览器插件调用的长期接口。

我可以在你授权和你看得见的情况下帮你操作电脑，比如帮你打开扩展页、加载插件、检查哪里点错了。但插件以后自己运行时，还是需要 DeepSeek API 或者你自己的后端接口。

## 图片题怎么办

先点“搜题：框选并讲解”，把图片题那一块框出来。如果识别不准，再点“拍照/上传识别”，选择一张更清晰的图片。第一次可能会慢一点，因为要下载中文和英文 OCR 语言包。

## 关于检测和切屏

这个插件是普通学习浮窗，不是防检测工具。我不能也不会帮你做隐藏切屏、规避网页监控、绕过禁止粘贴限制这类功能。

如果是考试、测验、作业平台，请按老师或平台规则使用。平时练习、复习、核对思路，可以正常用。
